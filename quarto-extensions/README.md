# Quarto Extensions Skills

Skills for using parmsam's Quarto extensions.

## Skills

| Skill | Description |
|-------|-------------|
| [quarto-quiz](./quarto-quiz/) | Add interactive multiple-choice quiz slides to RevealJS presentations |
| [quarto-flashcards](./quarto-flashcards/) | Create interactive flip-card study presentations in RevealJS |
| [quarto-quizdown](./quarto-quizdown/) | Embed interactive quizzes in Quarto HTML documents using quizdown syntax |
| [quarto-excalidraw](./quarto-excalidraw/) | Embed an Excalidraw whiteboard canvas in RevealJS presentations |
| [quarto-github-corner](./quarto-github-corner/) | Add a GitHub corner SVG badge to Quarto HTML documents |
| [quarto-brainmade](./quarto-brainmade/) | Add the Brainmade (human-made) mark via shortcode |
| [quarto-tts](./quarto-tts/) | Text-to-speech narration for RevealJS using the Web Speech API |
| [quarto-subtitles](./quarto-subtitles/) | Live speech-recognition subtitles for RevealJS presentations |
| [quarto-speech](./quarto-speech/) | Voice-controlled slide navigation for RevealJS presentations |
| [quarto-webcam](./quarto-webcam/) | Embed a live webcam video feed in RevealJS slides |
