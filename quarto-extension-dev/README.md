# quarto-extension-dev

Skills for **creating** Quarto extensions — filters, shortcodes, custom formats, and distribution. These complement `quarto-extensions/`, which covers *using* extensions.

## Skills

| Skill | Trigger |
|---|---|
| `quarto-ext-overview` | "create a quarto extension", "what type of extension should I make", `_extension.yml` |
| `quarto-ext-lua` | writing filters, shortcodes, Lua scripts, Pandoc AST |
| `quarto-ext-formats` | custom output formats, journal articles, RevealJS plugins |
| `quarto-ext-distributing` | publishing to GitHub, `quarto add`, versioning, starter templates |
